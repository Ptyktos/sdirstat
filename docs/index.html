sdirstat

The directory tree as a web. Parallel scan, one reverse-pass size fold, then QDirStat cache, JSON, total size, or a self-contained interactive treemap report.

sdirstat /usr -o usr.html

Features

Zero runtime dependencies

The Rust backend is std-only, while the GUI is vanilla JavaScript with no CDN and offline reports.

Byte-exact disk usage

Uses allocated size from blocks, deduplicates hardlinks, and supports apparent byte size when you need it.

Multiple outputs

Emit QDirStat cache files, nested JSON, a grand total, or a self-contained HTML report from the same scan.

Interactive views

Browse a squarified treemap, sunburst rings, file-type stats, sortable tree rows, and breadcrumbs.

Desktop entry

Install a local Linux launcher with no root, run the web GUI, or use the native Tauri desktop app.

Fast large scans

Parallel walking and a single reverse fold keep large filesystem scans practical and predictable.
